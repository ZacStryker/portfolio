Technical Overview

This project uses a neural network trained on the MNIST dataset (28x28 grayscale handwritten digits) with a Flatten → Dense(128, ReLU) → Dense(64, ReLU) → Dense(10, Softmax) architecture.

The trained Keras model is converted to TensorFlow.js format and loaded directly in the browser, so all inference runs client-side with no server calls.

When you click Predict, the canvas drawing is scaled down to 28x28 pixels, normalized to [0, 1], and fed through the network to produce a probability distribution over digits 0–9.

The predicted digit is the class with the highest softmax output, displayed alongside a confidence percentage and per-class probability bars.
