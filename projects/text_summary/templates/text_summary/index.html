Technical Overview

This project provides both extractive and abstractive summarization powered by a Flask API backend.

Extractive methods (TextRank, LSA, Luhn) use the sumy library to score and select the most important sentences from the original text based on graph ranking, singular value decomposition, or word frequency analysis.

The abstractive method uses the facebook/bart-large-cnn transformer model via Hugging Face, which generates new sentences that capture the meaning of the input using beam search decoding.

Text is sent to the server via a POST request, processed by the selected algorithm, and returned with metrics including word count, compression ratio, and processing time.
